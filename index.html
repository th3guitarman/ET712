Musician - Developer - Artist

Aaeron Jhonson-Whyte

My name is Aaeron and I am an artist in more ways than one. For the past 10 years, music has been my life. As a guitarist and composer, I've always enjoyed using music to create art for every project.

However recently, I've been composing more code than music. I'd always been interested in computer logic, but since learning languages like C++, Java and more recently HTML/CSS/Javascript, I've glimpsed the unlimited potential and possibility availed to programmers. Employing code to solve problems is artistic and creative, and creating interesting design elements to capture attention is thought-provoking fun.

I look forward to every opportunity to take a new challenge and, essentially, create yet another new piece of art.
