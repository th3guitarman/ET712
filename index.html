About Me

Aaeron Jhonson-Whyte

Musician - Developer - Artist

My name is Aaeron and I am an artist in more ways than one. For the past 10 years, music has been my life. As a guitarist and composer, I've always enjoyed using music to create art for every project.

However recently, I've been composing more code that music. I'd always been interested in computer logic but since learning languages like CSS, Javascript, HTML, C++, and Java, I've seen the unlimited potential and many possibilities when creating code. Employing code to solve problems is artistic and creative. Creating interesting design elements to capture attention is thought provoking fun.

I look forward to every opportunity to take a new challenge and, essentially, create yet another new piece of art.
